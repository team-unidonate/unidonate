# Uni-Donate

Uni-Donate es una plataforma web y móvil orientada a estudiantes universitarios que permite donar y solicitar materiales académicos dentro del campus de forma segura, rápida y sostenible.

## Descripción

Uni-Donate surge como respuesta a un problema común en los entornos universitarios: la falta de un sistema seguro y organizado para reutilizar materiales académicos entre estudiantes.  
La plataforma permite que los usuarios publiquen y soliciten objetos de manera rápida, utilizando puntos oficiales de entrega dentro del campus y validación mediante correo institucional.

## Características

- Acceso exclusivo con correo institucional.
- Publicación guiada con foto, categoría y estado del material.
- Chat interno seguro para coordinar entregas.
- Filtros académicos por curso, categoría y tipo de recurso.
- Historial de donaciones y trazabilidad.
- Puntos oficiales de entrega dentro del campus.
- Prototipos completos para Web, Android e iOS.

## Proceso de Diseño

El proyecto se desarrolló bajo un enfoque centrado en el usuario, utilizando:

- Entrevistas semiestructuradas  
- Segmentación y User Personas  
- User Journey Mapping  
- Empathy Map  
- AS-IS y TO-BE Scenario Mapping  
- Lean UX Canvas  
- Impact Mapping  
- User Stories & Acceptance Criteria  
- Diseño UI/UX multiplataforma  
- Evaluaciones heurísticas  
- Auditorías cruzadas (realizada y recibida)  

---

## Validación

Se realizaron múltiples validaciones:

- Pruebas de usabilidad  
- Entrevistas de validación  
- Evaluaciones heurísticas  
- Auditorías internas y externas  
- Ajustes finales en la entrega TF1  

Los resultados confirmaron que Uni-Donate es seguro, accesible, usable y alineado con las necesidades reales de los estudiantes universitarios.
